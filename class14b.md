
# 8 SIMPLE CSS3 TRANSITIONS THAT WILL WOW YOUR USERS
1. Change color
```
.color:hover
{
        background:#53a7ea;
}
```
2. . Grow  
```
.grow:hover
{
        -webkit-transform: scale(1.3);
        -ms-transform: scale(1.3);
        transform: scale(1.3);
}
```
3. Shrink
```
.shrink:hover
{
        -webkit-transform: scale(0.8);
        -ms-transform: scale(0.8);
        transform: scale(0.8);
}
```
